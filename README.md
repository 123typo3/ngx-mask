### How To Use

**npm:**
* Install the package:

   ```shell
   npm install ng2-mask
   ```
* Import ng2-mask Module:

   ```javascript
   import {Ng2MaskModule} from 'ng2-mask'
   ```
* Import module dependency:

   ```javascript
      ...
      imports: [
      ...
       Ng2MaskModule
      ...
      ]
      ```
   ```

* Use ng2-mask directive:

   ```html
   <input type='text' mask='0000-00-00' />
   ```
