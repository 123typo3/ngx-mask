Angular2 Mask Plugin

An Angular2 Plugin to make masks on form fields and html elements. Examples asdsad



Documentation

Basic Usage Examples

      <input class="date" mask="00/00/0000">
      <input class="time" mask="00:00:00">
      <input class="date_time" mask="00/00/0000 00:00:00">
      <input class="cep" mask="00000-000">
      <input class="phone" mask="0000-0000">
      <input class="phone_with_ddd" mask="(00) 0000-0000">
      <input class="phone_us" mask="(000) 000-0000">
      <input class="mixed" mask="AAA 000-S0S">
      <input class="ip_address" mask="099.099.099.099">
