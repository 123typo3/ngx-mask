Angular2 Mask Plugin

An Angular2 Plugin to make masks on form fields and html elements.
Examples
Mask common case An case of date mask
Mask: 00/00/0000

FormControl: {{ form.value ? form.value : 'Empty' }}

NgModel: {{ dateModel ? dateModel : 'Empty' }}
Other common cases Other common cases that you could apply in your projects. Try your self. :D
Mask: 00/00/00 00:00:00
Mask: 00:00:00
Mask: AAA 000-S0S
Mask: (000) 000-0000
Masks and special characters You could define if you want special characters propagate to the model or not with the attribute specialCharacters.
Mask: 000.000.000-00

FormControl: {{ cpfFormControl.value ? cpfFormControl.value : 'Empty' }}

NgModel: {{ cpfModel ? cpfModel : 'Empty' }}
